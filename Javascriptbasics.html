JavaScript Basics


Javascript is a cross-plattform, object-oriented scripting language. Javascript is extremely popular for a variety of reasons. It has a small and lightweight language allowing maximum flexability for develers to take it in a bunch of different directions. Java script lives inside a host environment(a web browser Node server), it can be connected to the objects of these environments to provide progmatic control over them.

  • Variable Decleration JavaScript variables are containers for storing data values-imagine a cup you fill with coffe, the cup holds the coffe,a variable holds a value. All JavaScript variables must be identified with unique names. These unique names are called identifiers.var x;

  • Variable Assignment Assignment opperators assign values to JavaScript variables - our cup can now have coffe poured in it, giving our variable a value to hold. The = assignment operator assigns a value toa variable. var x = 10;
